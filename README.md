# homebrew-tap

Official Homebrew tap for [A.R.C. Framework CLI](https://github.com/arc-framework/arc-cli).

## Install

```bash
brew tap arc-framework/tap
brew install arc
```

Or in one command:

```bash
brew install arc-framework/tap/arc
```

## Upgrade

```bash
brew upgrade arc
```

## Verify

```bash
arc version
```

## About

**A.R.C.** (Agent Runtime Core) is a powerful CLI for managing and orchestrating AI infrastructure.

- Source: [arc-framework/arc-cli](https://github.com/arc-framework/arc-cli)
- Releases: [github.com/arc-framework/arc-cli/releases](https://github.com/arc-framework/arc-cli/releases)
- Issues: Please report bugs at the [main repo](https://github.com/arc-framework/arc-cli/issues)

## How it works

The formula in `Formula/arc.rb` is automatically generated and pushed here by [GoReleaser](https://goreleaser.com/) on every tagged release of `arc-cli`. Do not edit `Formula/arc.rb` manually.
